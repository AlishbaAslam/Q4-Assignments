# Personal Finance Tracker CLI

A professional CLI application for tracking expenses, income, budgets, and generating financial insights.

## Core Features

- **Transaction Management**: Add, list, and view your income and expenses.
- **Budget Management**: Set monthly budgets for different categories and track your spending against them.
- **Financial Analytics**: Get insights into your spending patterns, income sources, and savings.
- **Smart Assistant**: A suite of intelligent features to help you manage your finances proactively.

## Smart Assistant

The Smart Assistant provides you with intelligent recommendations and proactive alerts to help you stay on top of your finances.

### Features:

- **Daily Financial Check**: Get a daily summary of your spending, your remaining budget for the day, and any important alerts.
- **Smart Recommendations**: Receive personalized recommendations based on your spending habits, such as identifying overspending categories or suggesting savings strategies.
- **Spending Alerts**: Get notified about important financial events, such as approaching budget limits or large transactions.
- **Savings & Goals**: Identify savings opportunities and set financial goals to track your progress over time.

## How to Run the Application

1. Ensure you have Python 3.11+ and `uv` installed.
2. Run the application using the following command:

   ```bash
   python main.py
   ```

## Tech Stack

- **Language**: Python 3.11+
- **CLI Framework**: `questionary`
- **UI Library**: `rich`
- **Storage**: Plain text files
- **Package Manager**: `uv`
